 ## Technology Which I Used
   ***Backend***

 - REST API
 - Nodejs
 - Mongodb
 - Express
 - Some tweaks (ex: jwt, CLOUDINARY, STRIPE, SMTP )
   ***Frontend***
 
 
 - React Js
 - Redux

<br />

 ## Step to Start Project
 
**For Backend**
 1. `main file "type npm i"`
 1. `npm run prod`

**For Frontend**

 1. `cd frontend`
 2. `npm i`
 3. `npm run start`

**Some requirement**
 1. `node ver 16`

**For setup runnin**
`Setting config.env file on config folder`
